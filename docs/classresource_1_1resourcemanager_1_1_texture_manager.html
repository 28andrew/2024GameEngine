short tint = 0 )

Fetches a texture from the cache or loads it if it doesn't exist.

The texture will be a SDL_TEXTUREACCESS_STREAMING one to allow for direct pixel access. Params: bmpfilepath = the path to the bitmap file tint = the tint to apply to the texture (0-255 hue) Returns: a (cached) texture
